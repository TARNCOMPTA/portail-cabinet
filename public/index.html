Comptes espace pro impôts

              

                Ajoute un ou plusieurs comptes « espace professionnel » impots.gouv.fr. Chaque client est rattaché à un compte. ⚠️ À cause du
                captcha, une fenêtre s'ouvre à chaque session : si tu renseignes l'e-mail et le mot de passe ci-dessous, ils
                sont préremplis automatiquement et il ne te reste qu'à saisir la captcha puis cliquer sur Connexion. Le robot
                enchaîne ensuite tous les clients. Le mot de passe est stocké chiffré en local.

Clé API (MCP) admin

              

                Clé d'accès pour le serveur MCP / les accès programmatiques, à la place d'un mot de passe. Elle donne un accès complet : ne la partage pas. Tu
                peux la révoquer ou la régénérer à tout moment (les anciennes configurations cesseront alors de fonctionner).
              

              

                
              

              

                
                
                
              

            


            

              
Connecteur MCP — organisation admin

              

                Pour ajouter le portail comme connecteur personnalisé dans Claude (niveau organisation / claude.ai). Dans Claude : « Ajouter un
                connecteur » → colle l'URL, puis dans « Paramètres avancés » colle le Client ID et le Client Secret. La connexion te demandera de t'authentifier
                avec ton compte du portail (OAuth).
              

              

                
                
                
              

              

                
                
              

            


            

              
Intégration n8n — webhook & API admin

              

                Webhook sortant : à la fin de chaque récupération, le portail envoie le bilan (JSON) à l'URL ci-dessous — colle l'URL d'un
                déclencheur « Webhook » n8n. API entrante : n8n peut appeler toute l'API du portail avec un nœud HTTP Request et l'en-tête
                X-API-Key (la clé API ci-dessus) — endpoints décrits dans le fichier INTEGRATION-N8N.md du dépôt.
              

              

                
                
              

              

                
                
                
              

            


            

              
Mise à jour de l'application admin

              

                Met à jour le portail en un clic depuis la dernière version publiée (GitHub), sans passer par le serveur. L'application redémarre toute seule
                (quelques secondes d'interruption) ; les données (clients, documents, clés) ne sont pas touchées.
